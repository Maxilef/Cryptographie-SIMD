    -RSA
    basé sur le probleme de la factorisation

    alice genere un paire de clef :

        elle choisi deux entier p q premier aléatooirement

        elle clacule de produit N

        elle calcule  phi(N)

        elle choisi un entier e tel que 1<e<phi(N) et e premier avec N => pgcd(e,phi(N)) = 1

        elle calcule d tel que d*e = 1 mod(phi(N))

            e est publié c'est la clef publique

            d est concervé c'est la clef secréte

    bob chiffre le msg :

        c = M^e mod N

    alice dechiffre le msg :

        c^d mod N = (M^e)^d mod N
                = M^e-d mod N
                = M^(e-d mod phi(N)) mod N
                = M¹ mod phi(N) mod N
                = M
